# Spotify Spotlight

### Prerequisites

- `npm` or `yarn`

### Installation

1. Clone the repo
2. Install NPM packages: `npm install` or `yarn install`
3. Run the app: `npm dev` or `yarn dev`
4. Copy the `.env.example` file to `.env` using `mv .env.example .env`
5. Change the `CLIENT_ID` to your own Spotify credentials
